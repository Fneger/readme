-----------------------------------------将本地文件添加到远程仓库-------------------------------------------------
git init 
git clone '你的仓库url'

git add .
git commit -m '上传1'
git remote add origin '远程仓库url'
这时候直接push不行，需要：
git pull --rebase origin master
git push -u origin master
其中master为远程仓库主分支名称
